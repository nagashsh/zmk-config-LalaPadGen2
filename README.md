# zmk-config-LalaPadGen2

[LaLaPadGen2](https://github.com/ShiniNet/LaLaPadGen2) 向け個人キーマップ設定です。

## キーマップについて

**OS の入力レイアウトを JIS 配列に設定した環境での利用を前提としています。**  
記号キーの割り当ては JIS 配列における出力文字を基準に設定しています。

---
